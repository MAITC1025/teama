This is what is learned about CSS

CSS stands for Cascading Style Sheets. Styles can be Inline, Internal or External. I leaner that Inline styles will override
Internal and External styles. Internal styles will override External styles. On smaller web pages I have found that Inline and
External styles are useful. However, that is not the case on large web pages due to the amount of manual typing required. That
is where an External style sheet has the advantage. The External style sheet allows the developer to make changes to multiple
pages and line items with a single code or group of codes.
